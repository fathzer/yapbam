Yapbam : Hilfe

Automatische Sicherungskopien

Für die Sicherheit Ihrer Daten ist es unerlässlich, Sicherungskopien anzulegen.

Yapbam ermöglicht es Ihnen, komprimierte Kopien aller Dateien auf einem Speichermedium (Speicher-Stick, interne oder externe Festplatte) oder per FTP auf einem Server anzulegen.

Die zweite Möglichkeit ist sehr empfehlenswert, vor allem wenn sich der Server an einem anderen Ort als Ihr PC befindet (im Brand- oder Überschwemmungsfall können Ihre Daten nur so gerettet werden).

Die meisten Internet-Provider ermöglichen den Zugriff auf einen FTP-Server.
Besuchen Sie deren Support-Seiten oder Ihre bevorzugte Suchmaschine, um mehr darüber zu erfahren.
