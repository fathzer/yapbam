Yapbam Hilfe

Import

Yapbam ermöglicht Ihnen den Import von Daten, die von einem anderen Kontenverwaltungsprogramm oder einer Tabellenkalkulation (wie Excel oder Open Office) im Textformat gespeichert wurden.

Wie Yapbam Zahlungsvorgänge, Teilbeträge und den Anfangskontostand erkennt:

  • Eine Zeile, die einen Wert in der Datumsspalte enthält, wird immer als Zahlungsvorgang angesehen.
  • Andere Zeilen sind Teilbeträge, wenn sie direkt auf einen Zahlungsvorgang folgen, oder der Anfangs-Kontostand, wenn sie vor einem Zahlungsvorgang stehen.
  • Wenn die Kontoangabe in einer Zeile mit Anfangs-Kontostand fehlt, wird der Kontostand dem Standardkonto zugewiesen.
  • Wenn ein Konto mehrere Zeilen mit Anfangs-Kontostand enthält, bildet deren Summe den tatsählichen Anfangsstand.

Datumsformat:

Das Datumsformat hängt von der in Yapbam eingestellten Sprache und dem Land ab.
Es ist das "kurze" Standard-Datumsformat. Dies wäre in den USA z.B. das Format "MM/DD/AA"

Dezimalformat für Beträge:

Das Format für Beträge hängt von der in Yapbam eingestellten Sprache und dem Land ab.
Das Trennzeichen zwischen dem ganzzahligen Teil und den Dezimalstellen ist z.B. ein Punkt in den USA und ein Komma in Deutschland. Es kann durch eine benutzerdefinierte Trenn ersetzt werden.
Yapbam akzeptiert das Währungszeichen, wenn es in Betragsfeldern angegeben wird.
