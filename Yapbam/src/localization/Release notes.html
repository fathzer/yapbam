Version 0.3.8 (12/??/2009)

This version is a prerelease with some missing functionnalities and, probably, some bugs.
You will also find some GUI items without any tooltip.

Nevertheless it's already useful to manage your bank accounts (I use it for my own). Feel free to report any bug.

The next functionnalities I plan to add :

  • Something to edit payment modes ... (currently, you can add or remove them, but not edit). For now, you have to manually edit the XML file ... at your own risks.
  • The ability to compress and encrypt data files and store them over the Internet.
  • An auto-complete wizard in the "new transaction" dialog that would be able to complete the description as you type and to deduce other informations (especially categories).
  • Maybe the ability to deal with OFX data files ... If a lot of users ask for it.

New in this version

  • Introduce better filtering capabilities (on category, modes, etc ...)..
