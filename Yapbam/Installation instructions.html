Installation instructions

  1. You need to have a java 5 (or more) runtime installed.
    Under Windows, a simple way to have it is to download and install the package provided by Sun (http://java.sun.com/javase/downloads).

    Under Linux, the best way is to use the package management tool.
    For instance, for Ubuntu type the following command in a console window :
    sudo apt-get install sun-java6-jre.
  2. Copy the yapbam.jar file ... where you want on your hard disk.
  3. Double-click on the yapbam.jar ... it should launch YAPBAM
    It is possible that the OS launch another application ... for instance an archive manager. A right click "launch with Java SE" (the exact wording may depends on your environment), will solve the problem.
    Note : It's ok, this is ugly !!! In a future release, I will probably try to solve this with a small piece of C code.

Uninstall YAPBAM

Delete yapbam.jar file and the hidden file .yapbam (it contains the preferences) ... and, if you really want ... your data files.
