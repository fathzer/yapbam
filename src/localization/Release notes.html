Version  0.2.0 (work in progress)

This version is a preview with somme missing functionnalities and, probably, a lot of bugs.
You will also find a lot of GUI items without any tooltip.

Nevertheless it's already useful to manage some bank account (I currently use it for my own). Feel free to report any bug. Please be indulgent, it's still a preview ;-)

The next functionnalities I hope to add :
  1. The ability to manage periodic transactions.
  2. Some statistics or charts about categories.
  3. Maybe - The ability to compress and encrypt data files.
  4. Maybe - An auto-complete wizard in the "new transaction" dialog that would be able to complete the description as you type and to deduce other informations (especially categories).

New in this version

  • Add the "Administration" tab which allows you to manage accounts and categories.
  • The bank account creation dialog allows you to create payment method.
