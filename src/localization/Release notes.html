Version  0.1.14 (13/09/2009)

This version is a preview with somme missing functionnalities and, probably, a lot of bugs.
You will also find a lot of GUI items without any tooltip.

Nevertheless it's already useful to manage some bank account (I currently use it for my own). Feel free to report any bug. Please be indulgent, it's still a preview ;-)

The next functionnalities I hope to add :

  1. The ability to define and manage periodic transactions.
  2. Something to manage accounts, category ... (currently, you can add them, but not edit or remove what you've added). For now, you have to manually edit the XML file ... at your own risks.
  3. Some statistics or charts about categories.
  4. Maybe - The ability to compress and encrypt data files.
  5. Maybe - An auto-complete wizard in the "new transaction" dialog that would be able to complete the description as you type and to deduce other informations (especially categories).

New in this version

  • Bug fixed : When the preferred localization changed, the dates in the transaction table were still displayed using the initial localization.
  • Add a new menu item to check for updates.
