Installation instructions

  1. You need to have a java 6 (or more) runtime installed.
    Under Windows, a simple way to have it is to download and install the package provided by Sun (http://java.sun.com/javase/downloads).

    Under Linux, the best way is to use the package management tool.
    For instance, for Ubuntu type the following command in a console window :
    sudo apt-get install sun-java6-jre.

    2. You may also have openJDK already installed with your Linux distribution. Yapbam is, of course, compatible with this Java Virtual Machine.

  2. Extract the content of yapbam.zip file ... to a new created folder where you want on your hard disk, or on an USB key.
  3. Double-click on the yapbam.jar ... it should launch YAPBAM
    It is possible that the OS launch another application ... for instance an archive manager. A right click "launch with Java SE" (the exact wording may depends on your environment), will solve the problem.
    If you're on windows, double click on yapbam.bat, it should solve the problem.
    Note : It's ok, this is ugly !!! In a future release, I will probably try to solve this with a small piece of C code.

Upgrading Yapbam

  1. Quit Yapbam.
  2. Extract the content of the zip file to the Yapbam folder (the one you've create when you installed Yapbam).
  3. Restart Yapbam.

Uninstall Yapbam

  • Delete the Yapbam folder.
    Be aware that Yapbam maintains two hidden files in its folder : .yapbam (which contains the yapbam startup state) and .yapbampref (it contains the preferences). Deleting only the files you see in the folder will be not enough, you have to delete the folder.
  • ... and, if you really want ... your data files.
