Yapbam : Help

Import

Yapbam allows you to import data saved in text format with another account manager or in a spreasheet (like Excel or Open Office).

How Yapbam detects transactions, subtransactions and account's initial balance :

  • A line containing a value in the date column is always considered as a transaction.
  • Other lines are subtransactions, if they are after a transaction, or an account initial balance if are located before any transaction.
  • If the account column of a "initial balance" line is empty, the balance is attached to the default account.
  • If an account has many "initial balance" lines, the initial balance will be the sum of these lines.

Dates format :

The date format depends on Yapbam language and country.
This format is the "short" standard date format. For instance, in US, dates have the format "MM/DD/AA"

Amounts format :

The amount format depends on Yapbam language and country.
The default separator between integer and decimal part of the amount is, for instance, the point in US and a comma in France. It can be replaced by a custom separator.
Yapbam accepts that currency symbol is included or not in amount fields.
