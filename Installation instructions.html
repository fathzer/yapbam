How to install Yapbam ? 

Windows

  1. You need to have a java 6 (or more) runtime installed.
    The easiest way to have it is to download and install the package provided by Sun (http://java.sun.com/javase/downloads).
  2. Copy the content of this archive in a new folder on your hard disk.
  3. Double-click on the yapbam.exe application
    Under other OS, double-click on the yapbam.jat file
    It is possible that the OS launch another application ... for instance an archive manager. A right click "launch with Java SE" (the exact wording may depends on your environment), will solve the problem.

Linux and others

  1. You need to have a java 6 (or more) runtime installed.
    Under Linux, the best way is to use the package management tool.
    For instance, for Ubuntu type the following command in a console window :
    sudo apt-get install sun-java6-jre
    Note : You may also have openJDK already installed with your Linux distribution. Yapbam is, of course, compatible with this Java Virtual Machine.
    For other OS, please refer to your OS documentation in order to known how to obtain a Java Runtime.
  2. Copy the content of this archive in a new folder on your hard disk
  3. Double-click on the yapbam.jar file
    It is possible that the OS launch another application ... for instance an archive manager. A right click "launch with Java SE" (the exact wording may depends on your environment), will solve the problem.

How to update Yapbam ?

  1. Quit Yapbam.
  2. Unzip the content of this archive into your Yapbam directory (the one you've created during the installation).
  3. Restart Yapbam.

How to uninstall YAPBAM ?

Delete the folder you've created during the installation ... and, if you really want ... your data files.
